How to use JavaScript and the Twitch.tv API for to create a custom browsing experience

This guide will overview how to make use of the Twitch.tv API to expand on the browsing experience provided through Twitch's user interface.

Contents:

  2. Create a successful GET request with Twitch
  3. Use a web page to display results
  4. Perform multiple searches from one screen
  5. Search across different twitch content types

Set up Twitch account and obtain Client ID

You will need your own Client ID to establish web service communication with Twitch. Follow these steps to get your Client ID

  1. Go to the Twitch.tv home page. Log in if you have an account - if not sign up for a free account.
  2. Go to the Settings page, and then to the Connections tab. At the bottom of the page, click "Register your Application" and complete the form.
  3. Once completed the form will provide you with your Client ID.

Create a successful GET request with Twitch

Before setting up a web page, test to ensure you can successfully communicate with Twitch. Chrome's Advanced Rest Client works well for this purpose.
